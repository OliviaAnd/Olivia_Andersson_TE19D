Beskrivning av ord

Element

Element är kommandon som oftast har en start tag och en slut tag, och emellan är content. T.ex. är html tagen, root-elementet till hela html dokumentet.

HTML

ID

ID används för att ge ett unikt id till ett element, för att lätt kunna hitta den i style.css.

Class

Class har en egenskap att specifiera ett eller flera element i HTML. Den används även för att programmet ska veta vilket/vilka element

Meta

Meta är ett element som alltid är innanför head elementet. Den ger information om webbsidan som bara webbläsaren kan se.

Responsive

Responsive Web Design handlar om att göra webbsidor som är anpassade att se bra ut på flera olika enheter.

http

http står för Hyper Text Transfer Protocol. Det är kommunikation mellan datorer och servrar.

https

https står för Hyper Transfer Protocol Secure. Den krypterar transporten av data för http, vilket gör att vi t.ex. kan göra betalningsöverföringar på ett säkert sätt.

Compressed data

CDN

Content Delivery Network är servrar utspridda över hela världen, vilket gör att data tas från den närmaste servern när någon ska ladda ner något, t.ex. spel.
